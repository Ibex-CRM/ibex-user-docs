Ibex Dashboard User Documentation

This document provides a comprehensive guide to using the Ibex dashboard located at https://ibex.goatagency.com/dashboard (opens in a new tab). It covers key features, functionalities, and common workflows to help you effectively manage your campaigns and data.

1. Getting Started

  • Logging In: Access the Ibex dashboard by navigating to https://ibex.goatagency.com/dashboard (opens in a new tab) in your web browser. Enter your provided username and password, then click "Login". Contact your administrator if you have forgotten your credentials.
  • Dashboard Overview: Upon successful login, you'll be presented with the main dashboard. This provides a high-level overview of your key metrics and campaign performance. Familiarize yourself with the layout and available information.

Why use Nextra?

The main reason to use Nextra (opens in a new tab) and MDX (opens in a new tab) for the Ibex User documentation instead of tools like Notion (opens in a new tab), which we use for other purposes, is the support of React (opens in a new tab) components inside our Markdown files. This allows us to create custom components that are specific to our documentation.

At the same time we are still writing in Markdown which allows us to migrate everything to Notion (opens in a new tab) easily, albeit with some loss of functionality.

Also, everything is save in a github (opens in a new tab) repository, so we can easily make changes and collaborate on the documentation.
